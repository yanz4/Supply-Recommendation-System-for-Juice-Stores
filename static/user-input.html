User Input

This function is developed for regional managers. Store managers can the output for adjusting their marketing and storage strategy. The assumption is that regional difference of favourite taste, ingredients availability and other feasibility factors are reported to regional manager to make marketing strategy at each store.

We have built-in constraints as follows:

1.According to previous experience, each store has a fixed amount of foot traffic conversion rate, and a fixed amount of sales number of 10k cups per store, regardless of the products offered or marketing efforts.

2.Due to seasonal availability, total pineapple available for each store is 30kg.

3.Due to taste preference survey, national total sale of Pumpkin Cordial will not reach 20, Lemon Drop will not reach 30, Arthritis Soother will not reach 10 (unit: k cups).

4.Due to taste preference survey, national total sale of Force Field will be at least 30 (unit: k cups).

The input are as follows:

Input 1

Input 1: Please input the max possible sales for any product. According to historic data, no product is likely to reach 50k cups in sales in all stores combined. Please enter a number, and it will overwrite the default 50k sales ceiling. If there is no solid evidence on substantial growth of the franchise, please leave it blank. Max possible sale for any product next year is: (please enter number only)

Input 2

Input 2: Please consult with marketing team to determine maximum possible total sales of each product. If no value is entered, default value will be the number from the entered number above. Default for input 1 is 50. Input Product ID, max total sales (k cups), separated by commas, no space. Input another constraint after the previous one, separated by a comma, no space. Please input product ID as capitalized P followed by its ID, such as ‘P1’. Please input max total sales as number only.
